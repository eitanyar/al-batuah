# על בטוח — סביבת פיתוח מקומית

האתר עובד עכשיו במחשב שלך, לפני שמעלים אותו לאינטרנט.

## איך לפתוח את האתר

1. פתח/י טרמינל ב-WSL.
2. הדבק/י את הפקודות האלו, אחת בכל פעם:

```bash
cd /home/eitanya/al-batuah
npm run dev
```

3. פתח/י בדפדפן את:

```
http://localhost:5173
```

זהו. כל שינוי שתשמור/י ב-`index.html`, `style.css` או `script.js` יתעדכן בדפדפן אוטומטית.

## איך לעצור את האתר

חזור/י לטרמינל שבו רץ האתר ולחץ/י:

```text
Ctrl + C
```

## מהו כל קובץ

- `index.html` — הטקסט והמבנה של האתר.
- `style.css` — הצבעים, העיצוב והמובייל.
- `script.js` — תפריט המובייל, טאבים ואינטראקציות.
- `images/` — לוגו ותמונות.
- `test/` — בדיקות בסיסיות כדי לא לשבור את סביבת הפיתוח.

## בדיקה לפני שינוי גדול

```bash
npm test
```

אם כתוב `pass 2`, הכול תקין.

## חשוב: טופס יצירת הקשר

הטופס עדיין לא שולח מיילים. זה מכוון: הוא לא יציג הודעה שקרית שהפנייה נשלחה. נחבר אותו בהמשך ל-Cloudflare Worker ולחשבון Resend של הלקוח.

## מה נעשה עכשיו

נבצע שינויים קטנים ובטוחים: אתה אומר מה לשנות באתר, ואני משנה, מפעיל את האתר ובודק בדפדפן. לא נעלה דבר ל-Cloudflare עד שתהיה מרוצה מהגרסה המקומית.
